# 🤖 AI Sentiment Analytics Dashboard

An AI-powered web application that analyzes customer reviews using sentiment analysis and presents interactive dashboards with business insights, visualizations, and review history.

---

## 📌 Overview

The AI Sentiment Analytics Dashboard enables users to upload customer review datasets in CSV format, perform sentiment analysis, and visualize the results through an interactive dashboard. It helps businesses understand customer opinions and identify trends using AI-driven analytics.

---

## ✨ Features

- 📂 Upload CSV files containing customer reviews
- 🤖 AI-powered Sentiment Analysis
- 📊 Interactive Dashboard
- 📈 KPI Cards
- 🥧 Sentiment Pie Chart
- 📉 Sentiment Bar Chart
- 📅 Sentiment Trend Analysis
- ☁️ Word Cloud Visualization
- 🎯 Average Confidence Score
- 🔍 Search Reviews
- 🏷️ Filter Reviews by Sentiment

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/AI-Sentiment-Analyzer.git
```

Move into the project directory

```bash
cd AI-Sentiment-Analyzer
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
